## 1. 项目概览

### 1.1 项目定位

**Medical Copilot** 是一个基于 **LangGraph** 和 **RAG** 技术的多Agent医疗文书生成系统，通过医患对话自动生成结构化病历（SOAP格式）并进行智能质控。

### 1.2 核心功能

- ✅ **多Agent协作**: 使用LangGraph编排4个专业Agent
- ✅ **RAG检索增强**: 支持4种检索模式（关键词/向量/GraphRAG/LlamaIndex）
- ✅ **结构化生成**: 输出符合医疗标准的SOAP格式病历
- ✅ **智能质控**: 自动检测缺项、剂量冲突、禁忌症等
- ✅ **语音识别**: 集成阿里云DashScope语音转文本功能
- ✅ **前端界面**: Streamlit演示UI

### 1.3 技术栈

| 层级 | 技术选型 |
|------|---------|
| **后端框架** | FastAPI |
| **工作流编排** | LangGraph |
| **LLM接口** | OpenAI SDK (兼容DeepSeek/OpenRouter) |
| **向量数据库** | Chroma |
| **图数据库** | NebulaGraph (可选) |
| **检索框架** | LlamaIndex |
| **语音识别** | DashScope Qwen ASR |
| **前端界面** | Streamlit |
| **数据验证** | Pydantic |

def get_asr_service() -> ASRService:

    return _asr_service_instance

@router.post("/api/generate-emr")

    workflow: Annotated[MedicalCopilotWorkflow, Depends(get_workflow)],
):
    # workflow参数自动注入
    result = await workflow.run(...)
    return result

### 3.4 工作流编排 (`src/graph/workflow.py`)

#### LangGraph状态图

```python
class MedicalCopilotWorkflow:
    def __init__(self, retrieval_mode: str = None):
        # 1. 初始化4个Agent
        self.dialogue_agent = DialogueAgent()
        self.retrieval_agent = create_retrieval_strategy(mode)
        self.generation_agent = GenerationAgent()
        self.qa_agent = QAAgent()
        
        # 2. 构建工作流图
        self.graph = self._build_graph()
    
    def _build_graph(self):
        workflow = StateGraph(GraphState)
        
        # 添加节点
        workflow.add_node("extract_info", self._extract_info_node)
        workflow.add_node("retrieve_knowledge", self._retrieve_knowledge_node)
        workflow.add_node("generate_emr", self._generate_emr_node)
        workflow.add_node("qa_check", self._qa_check_node)
        workflow.add_node("revise_emr", self._revise_emr_node)
        
        # 设置入口
        workflow.set_entry_point("extract_info")
        
        # 添加边（顺序执行）
        workflow.add_edge("extract_info", "retrieve_knowledge")
        workflow.add_edge("retrieve_knowledge", "generate_emr")
        workflow.add_edge("generate_emr", "qa_check")
        
        # 添加条件边（分支执行）
        workflow.add_conditional_edges(
            "qa_check",
            self._should_revise,
            {"revise": "revise_emr", "end": END}
        )
        
        # 修改后再次质控
        workflow.add_edge("revise_emr", "qa_check")
        
        return workflow.compile()
```

#### 节点函数详解

##### 节点1: 对话解析 (`_extract_info_node`)

```python
async def _extract_info_node(self, state: GraphState) -> GraphState:
    """从对话中提取医疗信息"""
    conversation = state.get("conversation", [])
    
    # 调用DialogueAgent
    extracted_info = await self.dialogue_agent.extract(conversation)
    
    # 更新状态
    state["extracted_info"] = extracted_info
    state["error_message"] = None
    
    return state
```

**输入**: 对话历史  
**输出**: 结构化医疗信息（症状、病程、用药等）

##### 节点2: 知识检索 (`_retrieve_knowledge_node`)

```python
async def _retrieve_knowledge_node(self, state: GraphState) -> GraphState:
    """检索相关临床指南"""
    extracted_info = state.get("extracted_info", {})
    symptoms = extracted_info.get("symptoms", [])
    
    # 调用RetrievalAgent
    guidelines = await self.retrieval_agent.retrieve_by_symptoms(symptoms)
    
    # 更新状态
    state["retrieved_guidelines"] = guidelines
    
    return state
```

**输入**: 提取的症状列表  
**输出**: 相关临床指南

##### 节点3: 病历生成 (`_generate_emr_node`)

```python
async def _generate_emr_node(self, state: GraphState) -> GraphState:
    """生成SOAP格式病历"""
    patient_info = state.get("patient_info", {})
    medical_info = state.get("extracted_info", {})
    guidelines = state.get("retrieved_guidelines", [])
    
    # 调用GenerationAgent
    draft_emr = await self.generation_agent.generate(
        patient_info,
        medical_info,
        guidelines
    )
    
    # 更新状态
    state["draft_emr"] = draft_emr
    
    return state
```

**输入**: 患者信息 + 医疗信息 + 临床指南  
**输出**: SOAP草稿病历

##### 节点4: 质控检查 (`_qa_check_node`)

```python
async def _qa_check_node(self, state: GraphState) -> GraphState:
    """质控检查"""
    draft_emr = state.get("draft_emr")
    medical_info = state.get("extracted_info", {})
    
    # 调用QAAgent
    qa_report = await self.qa_agent.check(draft_emr, medical_info)
    
    # 更新状态
    state["qa_report"] = qa_report
    state["final_emr"] = draft_emr  # 默认使用草稿
    
    # 判断是否需要修改
    state["needs_revision"] = not qa_report.is_complete
    state["iteration_count"] = state.get("iteration_count", 0) + 1
    
    return state
```

**输入**: 草稿病历 + 医疗信息  
**输出**: 质控报告 + 是否需要修改标志

##### 节点5: 病历修改 (`_revise_emr_node`)

```python
async def _revise_emr_node(self, state: GraphState) -> GraphState:

    # 简化版：直接标记为完成
    # 完整版会调用LLM根据建议修改
    state["needs_revision"] = False
    
    return state
```

**输入**: 质控报告  
**输出**: 修改后的病历

def _should_revise(self, state: GraphState) -> str:
    """决定是否需要修改病历"""
    needs_revision = state.get("needs_revision", False)
    iteration_count = state.get("iteration_count", 0)

### 4.4 质控检查Agent (`src/agents/qa_agent.py`)

#### 功能职责

#### 检查维度

1. **完整性检查**
   - 主诉是否为空
   - 诊断是否明确
   - 治疗计划是否完整

2. **合理性检查**
   - 药物剂量是否合理
   - 是否有禁忌症冲突
   - 检查项目是否必要

3. **一致性检查**
   - 病史与诊断是否一致
   - 治疗与诊断是否匹配

#### 核心实现

```python
class QAAgent:
    async def check(
        self,
        emr: SOAPNote,
        medical_info: MedicalInfoExtraction
    ) -> QAReport:
        issues = []
        
        # 1. 完整性检查
        if not emr.subjective:
            issues.append({
                "type": "missing",
                "field": "subjective",
                "message": "主诉为空"
            })
        
        # 2. 合理性检查（调用LLM）
        reasonableness = await self._check_reasonableness(emr, medical_info)
        issues.extend(reasonableness)
        
        # 3. 计算评分
        score = self._calculate_score(issues)
        
        return QAReport(
            is_complete=len(issues) == 0,
            issues=issues,
            score=score
        )
```

### 8.1 GraphState定义 (`src/graph/state.py`)

```python
class GraphState(TypedDict):
    """工作流状态定义"""
    
    # 输入
    conversation: List[Dict[str, str]]  # 对话历史
    patient_info: Dict[str, Any]        # 患者信息
    
    # 中间状态
    extracted_info: MedicalInfoExtraction  # 提取的医疗信息
    retrieved_guidelines: List[Dict]       # 检索的指南
    
    # 输出
    draft_emr: Optional[SOAPNote]    # 草稿病历
    final_emr: Optional[SOAPNote]    # 最终病历
    qa_report: Optional[QAReport]    # 质控报告
    
    # 控制流
    iteration_count: int              # 迭代次数
    needs_revision: bool              # 是否需要修改
    error_message: Optional[str]      # 错误信息
    
    # 元数据
    timestamp: str                    # 时间戳
    session_id: str                   # 会话ID
```
